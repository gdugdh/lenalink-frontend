# LenaLink Frontend

## Запуск

```bash
docker-compose up
```

Откройте [http://localhost:3000](http://localhost:3000)

## Тестовые пользователи

- `user@example.com` / `password` - Обычный пользователь
- `employee@example.com` / `password` - Сотрудник компании (ООО "Пример")
- `accountant@example.com` / `password` - Бухгалтер компании (ООО "Пример", баланс: 500,000₽)
- `admin@example.com` / `password` - Администратор

## Команды

```bash
docker-compose up          # Запустить
docker-compose down        # Остановить
docker-compose up --build  # Пересобрать
```
